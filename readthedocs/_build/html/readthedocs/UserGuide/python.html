Python User Guide¶

1. Instal¶

We recommend using conda to prepare the environment, especially when running on distributed cluster.

conda create -n zoo python=3.7 # "zoo" is conda enviroment name, you can use any name you like.
conda activate zoo

1.1 Official Release¶

You can install the latest release version of Analytics Zoo as follows:

pip install analytics-zoo

Note: Installing Analytics Zoo will automatically install bigdl==0.12.1, pyspark==2.4.3, conda-pack==0.3.1 and their dependencies.
